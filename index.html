How to Apply CSS to HTML

To include an external CSS file in an HTML document, use the <link> element inside the <head> section of the HTML file. The href attribute in the <link> tag points to the location of the CSS file. For example: 

<link rel="stylesheet" href="styles.css">
For more details, visit this MDN article on how to structure CSS in HTML.

How to Use JavaScript in a Webpage

To link to an external JavaScript file in an HTML document, use the <script> tag with the src attribute. This tag is usually placed just before the closing </body> tag. For example: 

<script src="app.js"></script>
You can also write JavaScript directly within the HTML file using the <script> element, like this: 
<script> console.log('Hello, world!'); </script>
For more details, visit this MDN article that explains how to import JavaScript code into a webpage.
